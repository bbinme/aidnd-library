# Fonts

Place the following TTF font files here to enable the Aethermere design fonts:

- `Cinzel-Regular.ttf`   — from Google Fonts: https://fonts.google.com/specimen/Cinzel
- `Raleway-Regular.ttf`  — from Google Fonts: https://fonts.google.com/specimen/Raleway

If these files are absent, the client falls back to `Serif` (Cinzel) and `SansSerif` (Raleway).
